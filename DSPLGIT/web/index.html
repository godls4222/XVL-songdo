DEMO1- 절제 및 측정

시작:

참고: 왼쪽 상단의 도구 모음은 단면 및 측정을 제어합니다. 왼쪽 열은 측정용이고 단면에 대한 오른쪽 열입니다.

  1. 첫 번째 스냅샷 선택

  2. 3D 창에서 피스톤 중 하나를 선택

    • 부품이 3D 창에서 강조 표시됩니다.
    • 부품은 단면/측정 창(오른쪽 위)에 표시됩니다.

  3. 실린더의 직경을 측정하려면직경 공구를 선택합니다.

    • 원통의 모서리선택(면이 아님)
    • 차원이 나타나면 이 차원을 원하는 위치로 드래그하여 가독성을 제공합니다.

  4. 단면에 오른쪽열의 위쪽 단추를 선택합니다.

    • 수행하려는 단면 유형을 선택합니다.

 

DEMO 2- 연결된 부품 목록

시작:

  1. 이름 옆의 화살표를 클릭하여 부품 목록 정렬

  2. "Rear_SUS"(후면 하위 어셈블리) 선택

  3. 이제 격리 창에 하위 어셈블리가 표시됩니다.

  4. 하위 어셈블리의 모든 부품을 보려면 "자식 부품 표시"를 선택합니다.

    • 모델의 부품을 클릭하면 부품이 부품 목록에서 강조 표시되고 부품 창에 표시됩니다.

  5. 당신이 원하는 데모의 모든 부분을 탐구. 길을 잃으면 홈 버튼을 클릭하여 시작으로 돌아갑니다.

 

DEMO 3- 구조화 된 BOM 및 3D 스냅샷

시작:

  1. 부품 번호(오름차순)로 정렬

  2. 하단 케이스 선택(LTC-001)

  3. 부분은 다음과 같은 것입니다.

    • 일부 브라우저 패널에 표시
    • 3D 모델에서 강조 표시
    • 구조화 된 BOM에서 강조 표시

  4. 3D 창에서 상위 케이스를 선택하고 동일한 경우를 알 수 있습니다.

  5. 스냅샷은 사용자에게 특정 뷰를 표시하고 두 번째 스냅샷(클러치)을 선택하도록 미리 구성됩니다.

    • 하위 어셈블리의 부품
    • BOM의 부품
    • 창 3D

 

DEMO 4- 작업이 있는 애니메이션

시작:

  1. 홈 버튼을 클릭하고 이 버튼을 사용하여 탐색 한 후 시작으로 돌아갑니다.

  2. 프로세스 #1 선택

  3. 애니메이션을 보려면 재생을 선택합니다.
